Wie vorher beschrieben werden die Daten nur anonymisiert gepeichert und weiterverarbeitet. Hierzu erstelle bitte einen 10-stelligen Code nach dem folgenden Muster:

Beispiel Code (10-stellig):

1. Der zweite und der letzte Buchstabe des Vornamens deiner Mutter (bei „Elke“: „LE“). Ist dir der Name unbekannt, der zweite und der letzte Buchstabe des Vornamens deiner Großmutter; ist dir auch dieser Name unbekannt: XX.

2. Der zweite und der letzte Buchstabe des Vornamens deines Vaters (bei „Martin“: „AN“). Ist dir der Name unbekannt, der zweite und der letzte Buchstabe des Vornamens deinesGroßvaters; ist dir auch dieser Name unbekannt: XX.

3. Dein eigener Geburtstag (nur der Tag, zweistellig, z.B. „06“)

4. Der Geburtstag (zweistellig, s.o.) deiner Mutter, wenn unbekannt: deines Vaters, wenn unbekannt: deiner Großmutter, wenn unbekannt: deines Großvaters (Bsp.: Alle unbekannt: 00).

5. Der 3. und der letzte Buchstabe des Geburtsnamens deiner Mutter, bei Unkenntnis: deines eigenen Nachnamens (bei „Müller“ LR)

Der Code im obigen Beispiel lautet also: LE AN 06 00 LR

Bitte den eigenen Code nach obigen Muster in das Textfeld eintragen und die Eingabe mit Enter bestätigen. Notiere dir den Code, da du ihn noch einmal für die Einwilligungserklärung deiner Eltern brauchst.
